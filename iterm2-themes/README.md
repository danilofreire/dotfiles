# iTerm2 themes

This folder contains a few iTerm2 colour schemes I use.

## Included themes

- Gruvbox
- Monokai Pro
- Monokai Pro Octagon
- Monokai Pro Ristretto
- Nord
- Pale Night Hc

## How to install

1. Download the `.itermcolors` file you want.
2. Double-click it, or run `open <filename>.itermcolors` in your terminal. iTerm2 will import it automatically.
3. Go to **iTerm2 > Settings > Profiles > Colors > Color Presets** and select the theme.

## More themes

Hundreds of additional themes are available at <https://iterm2colorschemes.com>.
