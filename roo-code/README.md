# Roo Code

Configuration for [Roo Code](https://roo.ai/) VS Code extension.

## Files

- `mcp_settings.json` - MCP server definitions
- `custom_modes.yaml` - custom agent modes

## Install

Copy the files to the Roo Code settings directory:

```bash
mkdir -p ~/Library/Application\ Support/Code\ -\ Insiders/User/globalStorage/rooveterinaryinc.roo-cline/settings
cp mcp_settings.json ~/Library/Application\ Support/Code\ -\ Insiders/User/globalStorage/rooveterinaryinc.roo-cline/settings/
cp custom_modes.yaml ~/Library/Application\ Support/Code\ -\ Insiders/User/globalStorage/rooveterinaryinc.roo-cline/settings/
```
